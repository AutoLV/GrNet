Usage:

Step1: Place the AFEW Grassmannian data under the folder "./data/afew/". The AFEW Grassmannian data can be downloaded from http://www.vision.ee.ethz.ch/~zzhiwu/

Step2: Launch grnet_afew.m for a simple example.
